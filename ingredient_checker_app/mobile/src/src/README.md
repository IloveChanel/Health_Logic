# Ingredient Checker Mobile App

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Run tests:
   ```
   npm test
   ```
3. Start the Expo app:
   ```
   npx expo start
   ```

## Folder Structure
- `components/` - UI components
- `screens/` - App screens
- `theme/` - Colors and typography
- `i18n/` - Translations
- `services/` - API and utility services
- `tests/` - Unit and integration tests
